![icon](https://i.imgur.com/jaGnidi.png)
## Описание
Это REfI - Rebalance Edition from Ilya, одного из участников перевода мода AEA в AUE. Она сделана одним человеком и несёт сугубо его личное мнение касательно определённых (почти всех) юнитов. Кто-то, вроде расы BGC, подвергся серьёзным изменениям баланса, а кто-то, вроде EKE, только минорным численным правкам. Если вас интересует баланс мода и вы хотите предложить что-то своё, обращайтесь по этому дискорду - essence_of_a_child. Не забудьте подписаться, что вы касательно мода. Чтож, развлекайтесь!
